# Code of Conduct

## Our Commitment

First Step Technology LLC is committed to fostering a welcoming, professional, and respectful community for everyone participating in this project.

We believe collaboration is strengthened by treating others with courtesy, professionalism, and respect regardless of experience level, background, identity, or technical expertise.

---

## Expected Behavior

Participants are encouraged to:

- Be respectful and constructive.
- Assume good intentions.
- Provide helpful technical feedback.
- Welcome newcomers.
- Share knowledge openly.
- Keep discussions focused on improving the project.
- Respect differing technical opinions.
- Help create a positive learning environment.

---

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or discrimination.
- Personal attacks or insults.
- Trolling or inflammatory comments.
- Threats or intimidation.
- Publishing private or confidential information.
- Deliberate disruption of discussions.
- Spam or promotional content unrelated to the project.
- Any behavior that creates an unsafe or hostile environment.

---

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation
- Community interactions
- Other official project communication channels

---

## Enforcement

Project maintainers may take appropriate action in response to behavior that violates this Code of Conduct, including:

- Requesting changes in behavior.
- Editing or removing inappropriate content.
- Closing discussions or issues.
- Restricting or revoking participation when necessary.

Enforcement decisions will be made with the goal of maintaining a productive and respectful community.

---

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it to the project maintainers through the appropriate GitHub communication channels or the contact information provided on the project website.

---
