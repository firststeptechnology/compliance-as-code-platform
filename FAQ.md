# Frequently Asked Questions (FAQ)

## What is Compliance-as-Code?

Compliance-as-Code is the practice of representing cybersecurity and regulatory requirements in structured, machine-readable formats instead of static documents. This enables automation, AI-assisted analysis, DevSecOps integration, and repeatable compliance workflows.

---

## What does this repository contain?

This repository contains documentation, sample files, and educational resources related to Compliance-as-Code.

Commercial rule libraries developed by First Step Technology LLC are distributed separately under commercial licensing.

## Why use machine-readable JSON instead of PDFs?

Machine-readable formats make it easier to:

- Automate compliance checks
- Integrate controls into software
- Build dashboards
- Search and filter requirements
- Support DevSecOps pipelines
- Feed AI assistants and internal tools
