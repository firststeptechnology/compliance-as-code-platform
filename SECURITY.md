# Security Policy

Thank you for helping improve this project.

## Reporting a Vulnerability

If you discover a potential security issue related to this repository, documentation, or published examples, please report it privately.

Please do **not** create a public issue for sensitive vulnerabilities.

## Contact

Email: security@firststeptechnology.com

Website: https://firststeptechnology.com

## Scope

This repository primarily contains documentation, examples, and educational materials.

Commercial products are distributed separately and are not included in this repository.
