# Compliance-as-Code Use Cases

The First Step Technology LLC Compliance-as-Code Platform is designed to support organizations across cybersecurity, privacy, governance, risk management, cloud security, DevSecOps, and artificial intelligence.

---

# Enterprise Compliance

Organizations can use the platform to:

- Standardize compliance processes
- Automate control validation
- Centralize compliance knowledge
- Reduce manual assessments
- Improve audit readiness
- Support continuous compliance

---

# DevSecOps

Development teams can integrate Compliance-as-Code into CI/CD pipelines to:

- Validate compliance requirements
- Detect invalid rule objects
- Prevent configuration drift
- Automate security gates
- Generate compliance reports
- Improve deployment quality

---

# Cloud Security

Cloud teams can use machine-readable rule libraries to:

- Validate cloud configurations
- Support Infrastructure-as-Code reviews
- Assess cloud environments
- Document security baselines
- Improve governance consistency

Supported cloud environments include:

- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)
- Oracle Cloud Infrastructure (OCI)
