# Developer Contribution Guide

Welcome to the First Step Technology LLC Compliance-as-Code Platform.

This repository contains the public documentation, schemas, and examples for the Compliance-as-Code Platform.

Commercial rule libraries are not stored in this repository.

---

# Repository Goals

This repository exists to:

- Document supported frameworks
- Publish public JSON schemas
- Demonstrate integrations
- Provide developer examples
- Support API documentation
- Encourage community learning

---

---

# Coding Standards

Contributors should follow these principles:

- Use UTF-8 encoding
- Use descriptive filenames
- Prefer JSON over proprietary formats
- Follow semantic versioning
- Maintain backward compatibility whenever possible
- Keep examples concise
- Document new features

---

# JSON Standards

Example JSON should:

- Be properly formatted
- Validate against published schemas
- Use descriptive property names
- Avoid unnecessary nesting
- Remain vendor neutral where practical

---

# Documentation Standards

Documentation should:

- Use Markdown
- Include headings
- Include examples
- Explain purpose before implementation
- Link to related documentation when appropriate

---

# Pull Requests

Recommended pull requests include:

- Documentation improvements
- Typographical corrections
- Example enhancements
- Schema improvements
- Developer tooling
- Integration examples

---

# Excluded Content

Do not submit:

- Proprietary framework content
- Licensed publications
- Confidential information
- Customer data
- Security credentials
- Internal commercial rule libraries
