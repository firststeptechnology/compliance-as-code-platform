# Frequently Asked Questions (FAQ)

Welcome to the Frequently Asked Questions for the First Step Technology LLC Compliance-as-Code Platform.

---

# What is Compliance-as-Code?

Compliance-as-Code is the practice of representing regulatory, security, privacy, and governance requirements in structured, machine-readable formats that can be consumed by software, automation platforms, AI systems, and DevSecOps pipelines.

---

# What does this repository contain?

This public repository contains:

- Documentation
- JSON Schemas
- Integration Guides
- Example JSON Objects
- Sample Pipelines
- API Documentation
- Developer Resources

Commercial rule libraries are not included.

---

# Which frameworks are supported?

Current commercial products include:

- CMMC Level 1
- CMMC Level 2
- NIST SP 800-171 Rev. 2
- NIST SP 800-53 Rev. 5 Core Controls
- NIST Cybersecurity Framework (CSF) 2.0
- NIST AI Risk Management Framework (AI RMF) 1.0
- HIPAA Security Rule
- CCPA / CPRA

---

# Are the complete rule libraries available?

No.

The commercial rule libraries are proprietary products of First Step Technology LLC and are licensed separately.
