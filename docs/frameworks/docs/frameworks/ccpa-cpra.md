# CCPA / CPRA Compliance-as-Code Rule Library

> Machine-readable implementation of key California Consumer Privacy Act (CCPA) and California Privacy Rights Act (CPRA) requirements.

## Overview

The **CCPA / CPRA Compliance-as-Code Rule Library** converts California privacy requirements into structured JSON for privacy automation, governance, compliance monitoring, AI-assisted workflows, and enterprise privacy programs.
