# Compliance-as-Code Platform Roadmap

> Public product roadmap for the First Step Technology LLC Compliance-as-Code Platform.

This roadmap outlines the planned evolution of the platform. It reflects current development priorities and is intended to provide transparency into the direction of the project. Items may change as customer needs, regulatory updates, and platform capabilities evolve.

---

# Vision

Build a modern Compliance-as-Code platform that enables organizations to transform regulatory requirements into machine-readable assets for AI, automation, DevSecOps, Governance, Risk & Compliance (GRC), and continuous compliance programs.

---

# Current Platform Status

## Available

- ✅ CMMC Level 1 Rule Library
- ✅ CMMC Level 2 Rule Library
- ✅ NIST SP 800-171 Rev. 2 Rule Library
- ✅ NIST SP 800-53 Rev. 5 Core Controls Rule Library
- ✅ NIST Cybersecurity Framework (CSF) 2.0 Rule Library
- ✅ NIST AI RMF 1.0 Rule Library
- ✅ HIPAA Security Rule Rule Library
- ✅ CCPA / CPRA Rule Library
